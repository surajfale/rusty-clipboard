# Bootstrap Plan

## Steps

1. workspace-setup — Initialize Cargo workspace with `clipd` and `clipctl` crates, shared dependencies, and base README structure.
2. clipd-skeleton — Scaffold `clipd` main executable with Tokio loop, Windows clipboard listener stubs, SQLite persistence module placeholders, and IPC server skeleton.
3. clipctl-skeleton — Scaffold `clipctl` TUI using ratatui + crossterm, IPC client stub, navigation/search handlers, and paste abstraction placeholder.
4. docs-and-config — Draft technical design summary, build/run instructions, config template, keybinding table, risk log, and short roadmap.
